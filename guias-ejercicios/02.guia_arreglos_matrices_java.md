# Arreglos y matrices en Java

> Guía teórico-práctica para el curso de Programación Orientada a Objetos — Ingeniería de Sistemas

---

## 1. ¿Qué es un arreglo?

Un **arreglo** (también llamado *array* o *vector*) es una estructura de datos que permite almacenar **múltiples valores del mismo tipo** bajo un solo nombre. Cada valor ocupa una posición identificada por un **índice numérico** que comienza en `0`.

Imagina una fila de casillas numeradas en una estantería: cada casilla guarda un dato y puedes acceder a cualquiera de ellas directamente si conoces su número.

```
Índice:  [0]   [1]   [2]   [3]   [4]
Valor:    10    25    8     47    3
```

### ¿Por qué usar arreglos?

Sin arreglos, para guardar 5 calificaciones necesitarías 5 variables separadas:

```java
int nota1 = 10;
int nota2 = 25;
int nota3 = 8;
int nota4 = 47;
int nota5 = 3;
```

Con un arreglo, lo haces en una sola línea y puedes recorrer todos los valores con un ciclo.

---

## 2. Arreglos en Java

### 2.1 Declaración

```java
tipo[] nombreArreglo;
```

Ejemplos:

```java
int[] notas;
double[] precios;
String[] nombres;
```

### 2.2 Creación (asignación de memoria)

```java
nombreArreglo = new tipo[tamaño];
```

También se puede declarar y crear en una sola línea:

```java
int[] notas = new int[5];   // arreglo de 5 enteros, todos inician en 0
```

> **Importante:** en Java, los arreglos tienen **tamaño fijo** desde el momento en que se crean. No puedes agregar ni eliminar posiciones después.

### 2.3 Inicialización con valores

Puedes asignar los valores directamente al declarar el arreglo:

```java
int[] notas = {10, 25, 8, 47, 3};
```

Java cuenta los elementos automáticamente y determina el tamaño.

### 2.4 Acceso a elementos

Se accede a cada elemento usando su índice entre corchetes:

```java
System.out.println(notas[0]);   // imprime 10
System.out.println(notas[3]);   // imprime 47

notas[2] = 99;                  // modifica el elemento en la posición 2
```

> **Cuidado:** si intentas acceder a un índice fuera del rango (por ejemplo `notas[10]` en un arreglo de tamaño 5), Java lanzará un error `ArrayIndexOutOfBoundsException`.

### 2.5 Propiedad `length`

Para saber el tamaño de un arreglo, usa `.length`:

```java
System.out.println(notas.length);   // imprime 5
```

## 3. Ejemplo completo: arreglo en Java

El siguiente programa guarda 5 notas de un estudiante, las muestra en pantalla y calcula el promedio.

```java
public class EjemploArreglo {

    public static void main(String[] args) {

        // Declaración e inicialización del arreglo
        int[] notas = {85, 90, 78, 92, 88};

        // Mostrar todos los elementos
        System.out.println("Notas del estudiante:");
        for (int i = 0; i < notas.length; i++) {
            System.out.println("  Nota " + (i + 1) + ": " + notas[i]);
        }

        // Calcular el promedio
        int suma = 0;
        for (int i = 0; i < notas.length; i++) {
            suma = suma + notas[i];
        }

        double promedio = (double) suma / notas.length;
        System.out.println("Promedio: " + promedio);
    }
}
```

**Salida esperada:**

```
Notas del estudiante:
  Nota 1: 85
  Nota 2: 90
  Nota 3: 78
  Nota 4: 92
  Nota 5: 88
Promedio: 86.6
```

### Versión con `for-each`

Java permite recorrer arreglos con una sintaxis más compacta cuando no necesitas el índice:

```java
for (int nota : notas) {
    System.out.println(nota);
}
```

Esto se lee como: *"para cada `nota` dentro de `notas`, ejecuta el bloque"*.

## 4. ¿Qué es una matriz?

Una **matriz** es un arreglo de dos dimensiones: tiene **filas** y **columnas**. Puedes visualizarla como una tabla donde cada celda se identifica con dos índices: `[fila][columna]`.

```
         Col 0   Col 1   Col 2
Fila 0  [  1  ] [  2  ] [  3  ]
Fila 1  [  4  ] [  5  ] [  6  ]
Fila 2  [  7  ] [  8  ] [  9  ]
```

Las matrices son útiles cuando los datos tienen una estructura de tabla natural: tableros de juego, mapas de píxeles, tablas de notas por estudiante y materia, entre otros.

---

## 5. Matrices en Java

### 5.1 Declaración y creación

```java
tipo[][] nombreMatriz = new tipo[filas][columnas];
```

Ejemplo: matriz de 3 filas y 3 columnas:

```java
int[][] tabla = new int[3][3];
```

### 5.2 Inicialización con valores

```java
int[][] tabla = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};
```

Cada par de llaves internas representa una fila.

### 5.3 Acceso a elementos

```java
System.out.println(tabla[0][2]);   // fila 0, columna 2 → imprime 3
System.out.println(tabla[2][1]);   // fila 2, columna 1 → imprime 8

tabla[1][1] = 99;                  // modifica el elemento en fila 1, columna 1
```

### 5.4 Dimensiones de la matriz

```java
int filas    = tabla.length;       // número de filas
int columnas = tabla[0].length;    // número de columnas de la primera fila
```

## 6. Ejemplo completo: matriz en Java

El siguiente programa crea una tabla de multiplicar del 1 al 4 y la imprime con formato.

```java
public class EjemploMatriz {

    public static void main(String[] args) {

        int filas    = 4;
        int columnas = 4;

        // Crear y llenar la matriz
        int[][] tabla = new int[filas][columnas];

        for (int i = 0; i < filas; i++) {
            for (int j = 0; j < columnas; j++) {
                tabla[i][j] = (i + 1) * (j + 1);
            }
        }

        // Imprimir la matriz
        System.out.println("Tabla de multiplicar (1 al 4):");
        System.out.println();

        for (int i = 0; i < filas; i++) {
            for (int j = 0; j < columnas; j++) {
                System.out.printf("%4d", tabla[i][j]);
            }
            System.out.println();
        }
    }
}
```

**Salida esperada:**

```
Tabla de multiplicar (1 al 4):

   1   2   3   4
   2   4   6   8
   3   6   9  12
   4   8  12  16
```

> `System.out.printf("%4d", valor)` imprime el número con 4 espacios de ancho, lo que alinea las columnas correctamente.

## 7. Diferencias clave entre arreglo y matriz

| Característica | Arreglo (1D) | Matriz (2D) |
|---|---|---|
| Dimensiones | Una (índice único) | Dos (fila y columna) |
| Declaración | `int[] a` | `int[][] m` |
| Creación | `new int[n]` | `new int[f][c]` |
| Acceso | `a[i]` | `m[i][j]` |
| Recorrido | Un ciclo `for` | Dos ciclos `for` anidados |
| Uso típico | Listas, secuencias | Tablas, cuadrículas, tableros |

---

## 8. Errores frecuentes

| Error | Causa | Solución |
|---|---|---|
| `ArrayIndexOutOfBoundsException` | Se accede a un índice fuera del rango | Verificar que el índice esté entre `0` y `length - 1` |
| `NullPointerException` | El arreglo fue declarado pero nunca creado con `new` | Asegurarse de usar `new tipo[tamaño]` antes de acceder |
| Todos los valores son `0` o `null` | El arreglo fue creado pero no inicializado con datos | Asignar valores a cada posición después de crearlo |
| Confundir `length` con `length()` | `length` es propiedad de arreglos; `length()` es método de `String` | Usar `arreglo.length` (sin paréntesis) para arreglos |

---

## 9. Resumen

- Un **arreglo** almacena múltiples valores del mismo tipo con un solo índice.
- Una **matriz** extiende esa idea a dos dimensiones: filas y columnas.
- Ambos tienen **tamaño fijo** en Java: se define al crearlos con `new`.
- Los índices **siempre comienzan en `0`**.
- Para recorrer un arreglo se usa un ciclo `for`; para una matriz, dos ciclos `for` anidados.
