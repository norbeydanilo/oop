# Crear y ejecutar tu primer archivo Java en VSCode

> Guía paso a paso — desde abrir el editor hasta ver el resultado en pantalla

---

## Requisito previo: instalar el JDK

Antes de comenzar, necesitas tener instalado el **JDK (Java Development Kit)**. Descárgalo desde [adoptium.net](https://adoptium.net) e instálalo. VSCode lo detectará automáticamente.

---

## Paso 1 — Abrir Visual Studio Code

Busca VSCode en tu menú de inicio o aplicaciones y ábrelo. La primera vez verás una pantalla de bienvenida.

---

## Paso 2 — Instalar la extensión de Java

1. Ve al panel de extensiones con `Ctrl+Shift+X`.
2. Busca **Extension Pack for Java** (publicada por Microsoft).
3. Haz clic en **Instalar**.

Esta extensión incluye soporte de sintaxis, depurador y ejecutor de Java.

> **Nota:** si el JDK no está instalado, la extensión te avisará con un mensaje de error al abrirla.

---

## Paso 3 — Abrir o crear una carpeta de trabajo

1. Ve a **Archivo → Abrir carpeta** (o presiona `Ctrl+K` luego `Ctrl+O`).
2. Crea una carpeta nueva en tu equipo, por ejemplo `MisClasesJava`.
3. Selecciónala y ábrela. VSCode la usará como espacio de trabajo.

---

## Paso 4 — Crear un archivo nuevo `.java`

En el explorador de archivos (panel izquierdo), haz clic en el ícono de **nuevo archivo** e ingresa el nombre:

```
HolaMundo.java
```

> **Regla clave:** el nombre del archivo **debe coincidir exactamente** con el nombre de la clase, incluyendo mayúsculas. Java es sensible a mayúsculas (`HolaMundo` es diferente de `holamundo`).

---

## Paso 5 — Escribir el código Java

Escribe (o pega) el siguiente código dentro del archivo. Es el programa mínimo funcional en Java:

```java
public class HolaMundo {

    public static void main(String[] args) {
        System.out.println("¡Hola, mundo!");
    }

}
```

**¿Qué hace cada parte?**

| Línea | Descripción |
|---|---|
| `public class HolaMundo` | Define la clase principal |
| `public static void main(String[] args)` | Punto de entrada del programa |
| `System.out.println(...)` | Imprime texto en la consola |

---

## Paso 6 — Guardar el archivo

Presiona `Ctrl+S` para guardar. El punto naranja en la pestaña del archivo desaparecerá, indicando que el archivo está guardado.

---

## Paso 7 — Ejecutar el programa desde la interfaz

Con el archivo abierto, tienes dos opciones:

**Opción A (recomendada):** verás un botón **▷ Run** justo encima del método `main`. Haz clic en **Run Java**.

**Opción B:** haz clic derecho en cualquier parte del editor y selecciona **"Run Java"** en el menú contextual.

**Atajo de teclado:** `Ctrl+F5` ejecuta el programa sin depurador.

---

## Paso 8 — Ver el resultado en la terminal

VSCode abrirá automáticamente el panel **Terminal** en la parte inferior y mostrará la salida del programa:

```
¡Hola, mundo!
```

¡Listo! Tu primer programa Java está funcionando.

---

## Errores frecuentes y cómo resolverlos

| Error | Causa | Solución |
|---|---|---|
| `class HolaMundo is public, should be declared in a file named HolaMundo.java` | El nombre del archivo no coincide con la clase | Asegúrate de que el archivo se llame exactamente igual que la clase |
| `Cannot find or load main class` | JDK no encontrado o mal configurado | Instala el JDK desde [adoptium.net](https://adoptium.net) y reinicia VSCode |
| `';' expected` | Falta punto y coma al final de una instrucción | Cada línea de código en Java termina con `;`. VSCode marcará el error con subrayado rojo |
| El botón `Run` no aparece | La extensión no está instalada o no se activó | Verifica que **Extension Pack for Java** esté instalada y el archivo tenga extensión `.java` |

```

Instalar Extension Pack for Java
     ↓

     ↓
Crear archivo HolaMundo.java
     ↓
Escribir el código y guardar (Ctrl+S)
     ↓

     ↓

```
