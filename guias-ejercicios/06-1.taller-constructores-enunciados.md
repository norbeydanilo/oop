# Taller: Constructores y Sobrecarga de Métodos

Hasta ahora creábamos objetos vacíos y luego llamábamos varios setters para asignarles datos. En este taller aprenderás a usar **constructores** para inicializar un objeto desde el momento en que se crea.

Un constructor es un método especial que:
- Tiene el mismo nombre que la clase
- No tiene tipo de retorno
- Se ejecuta automáticamente cuando se usa `new`

Cuando una clase tiene **varios constructores** con diferentes parámetros, eso se llama **sobrecarga**. Java decide cuál usar según los argumentos que se le pasen.

```java
// Ejemplo con la clase Persona:
Persona p1 = new Persona();                  // constructor vacío
Persona p2 = new Persona("Ana");             // constructor con nombre
Persona p3 = new Persona("Ana", 25);         // constructor completo
```

## Ejercicio 1 — Libro

Modifica la clase `Libro` del taller anterior agregando **tres constructores**:

- **Constructor vacío**: no recibe parámetros. Asigna `disponible = true` por defecto.
- **Constructor con título y autor**: recibe `titulo` y `autor`. El libro queda disponible por defecto.
- **Constructor completo**: recibe `titulo`, `autor` y `disponible`.

Los getters, setters y métodos (`mostrarInfo`, `prestar`, `devolver`) se mantienen igual.

En `App.java` crea tres objetos usando cada uno de los constructores y llama a `mostrarInfo()` en cada uno.

## Ejercicio 3 — Depósito de agua

Modifica la clase `DepositoAgua` del taller anterior agregando **tres constructores**:

- **Constructor vacío**: no recibe parámetros.
- **Constructor con capacidad**: recibe solo `capacidad`. El `volumenActual` inicia en 0 y `depositoDesborde` en `null`.
- **Constructor completo**: recibe `capacidad` y `volumenActual`.

En `App.java` crea dos depósitos usando constructores (no setters), conéctalos con `setDepositoDesborde()` y prueba el desbordamiento.

> **Nota:** `depositoDesborde` no se pasa por constructor porque el objeto al que apunta se crea por separado. La conexión se hace después con el setter, igual que antes.
