# Guía: Crear un proyecto Java en VSCode

## Requisitos previos

Antes de comenzar, asegúrate de tener instalado:

- **Java JDK 11 o superior** — [descargar en adoptium.net](https://adoptium.net/)
- **Visual Studio Code** — [descargar en code.visualstudio.com](https://code.visualstudio.com/)

---

## Paso 1 — Instalar el Extension Pack de Java

1. Abre VSCode.
2. Ve al panel de **Extensiones** (ícono de bloques en la barra lateral, o `Ctrl+Shift+X`).
3. Busca: `Extension Pack for Java` (publicado por Microsoft).
4. Haz clic en **Install**.

Este paquete incluye todo lo necesario: soporte de lenguaje, depurador, gestión de proyectos y pruebas.

---

## Paso 2 — Verificar que Java está reconocido

1. Abre la terminal integrada (`Ctrl+ñ` o `View > Terminal`).
2. Ejecuta:

```bash
java -version
```

Deberías ver algo como:

```
openjdk version "21.0.2" 2024-01-16
```

Si aparece un error, revisa que el JDK esté instalado y que la variable de entorno `JAVA_HOME` esté configurada correctamente.

---

## Paso 3 — Crear el proyecto

1. Abre la **Paleta de comandos** con `Ctrl+Shift+P`.
2. Escribe y selecciona:

```
Java: Create Java Project
```

3. Elige el tipo de proyecto. Para comenzar, selecciona **No build tools** (sin Maven ni Gradle).

4. Selecciona la carpeta donde se guardará el proyecto.

5. Escribe el nombre del proyecto, por ejemplo: `MiProyectoJava`.

VSCode creará automáticamente la estructura de carpetas.

---

## Paso 4 — Estructura del proyecto

Después de crearlo, verás esta estructura en el explorador:

```
MiProyectoJava/
├── src/
│   └── App.java        ← archivo principal generado automáticamente
├── lib/                ← aquí van las dependencias externas (.jar)
└── README.md
```

---

## Paso 5 — Explorar el archivo principal

Abre `src/App.java`. Verás el código generado:

```java
public class App {
    public static void main(String[] args) throws Exception {
        System.out.println("Hello, World!");
    }
}
```

Este es el punto de entrada del programa. El método `main` es el primero que se ejecuta.

---

## Paso 6 — Ejecutar el programa

Tienes dos opciones:

**Opción A — Botón Run:**
Haz clic en el botón ▶ **Run** que aparece encima del método `main`.

**Opción B — Terminal:**

```bash
cd MiProyectoJava
javac src/App.java -d bin
java -cp bin App
```

La salida en la terminal debe ser:

```
Hello, World!
```

---

## Paso 7 — Agregar un nuevo archivo Java

Por ahora solo necesitas saber esto: **cada clase va en su propio archivo `.java`**, y el nombre del archivo debe ser igual al nombre de la clase.

Para crear un archivo nuevo:

1. Haz clic derecho sobre la carpeta `src` en el explorador.
2. Selecciona **New File**.
3. Escribe el nombre del archivo, por ejemplo: `Saludo.java`.
4. Dentro del archivo escribe lo mínimo para que funcione:

```java
public class Saludo {

}
```

Por ahora déjalo así. A lo largo del curso irás aprendiendo qué poner dentro de la clase. Lo importante es que ya sabes cómo crear el archivo y dónde guardarlo.

---

## Paso 8 — Usar el depurador (Debugger)

1. Haz clic en el margen izquierdo de cualquier línea para agregar un **breakpoint** (punto de interrupción). Aparece un círculo rojo.
2. Abre el panel **Run and Debug** (`Ctrl+Shift+D`).
3. Haz clic en **Run and Debug** o presiona `F5`.
4. El programa se detendrá en el breakpoint y podrás inspeccionar el valor de las variables en tiempo real.

---

## Resumen de atajos útiles

| Acción | Atajo |
|---|---|
| Paleta de comandos | `Ctrl+Shift+P` |
| Terminal integrada | `Ctrl+ñ` |
| Ejecutar programa | `F5` |
| Autocompletar código | `Ctrl+Space` |
| Formatear archivo | `Shift+Alt+F` |
| Buscar en el proyecto | `Ctrl+Shift+F` |
| Renombrar variable/clase | `F2` |

---

## Próximos pasos recomendados

- **Maven o Gradle** — gestores de dependencias para proyectos más grandes.
- **JUnit** — para escribir pruebas unitarias (VSCode tiene soporte nativo con el Extension Pack).
- **GitHub Copilot** — autocompletado con IA, disponible como extensión en VSCode.
